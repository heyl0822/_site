Ok, back to the topic, I decide to use jekll mainly because of Aerotwist’s suggestion and want to learn the structure of building a static website.

Quickstart

I first download and follow the steps shown on line:

~ $ gem install jekyll
~ $ jekyll new learn-blog
~ $ cd learn-blog
~/my-awesome-site $ jekyll serve

I add a file with format YYYY-MM-DD-name-of-post.ext under _posts directory.

Editing Format

I change the blog title inside _config.yml file, and can always run jekll serve to rebuild the whole site.
